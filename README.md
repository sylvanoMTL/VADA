# VADA

**V**isual **A**larm **D**evice **A**nalyzer — Coverage volume measurement and analysis tool for EN 54-23 fire safety compliance.

## About

VADA is a standalone Windows application for fire safety engineers to measure and analyze coverage volumes of visual alarm devices (VADs) in accordance with EN 54-23.

This repository hosts official releases only. Source code is maintained privately.

## Download

Go to the [Releases](https://github.com/sylvanoMTL/VADA/releases) page and download the latest `.exe` installer.

### System Requirements

- Windows 10 or later
- [MATLAB Runtime R2025a](https://www.mathworks.com/products/compiler/matlab-runtime.html) (free, installed automatically if needed)

## Installation

1. Download the installer from the latest release
2. Run the `.exe` — it will install MATLAB Runtime if not already present
3. Launch VADA from the Start Menu or desktop shortcut

The application checks for updates automatically on startup.
